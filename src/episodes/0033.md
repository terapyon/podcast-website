# Note

<p>辻さんの自己紹介</p>
<p>1975年生まれの東京都足立区出身。小学生の頃MSX2を父に買ってもらってからずっと、コンピュータとプログラミングに興味があります。仕事柄、生命科学も詳しいですが、最近は違う分野も見てみたいと思ったりしています。美味しいものとお酒が好き。</p>
<p><br></p>
<ul>
 <li>Pythonスタートブック(技術評論社 <a href="https://amzn.to/3iXahKS" rel="noreferrer nofollow noopener" target="_blank">https://amzn.to/3iXahKS</a> )の執筆秘話</li>
 <li>Start Python Clubコミュニティ( <a href="https://startpython.connpass.com/" rel="noreferrer nofollow noopener" target="_blank">https://startpython.connpass.com/</a> )でみんなのPython勉強会を約5年やっているが、最初は主催者の阿久津さんが辻さんを探して始まった</li>
  <li>オフラインイベントの立ち話をしたいのと世界各国の学会で刺激を受けたい</li>
  <li>Python Charity Talks in JapanでPSFが身近に感じてくれた</li>
  <li>一般社団法人Pythonエンジニア育成推進協会(Python ED)はPSFブロンズレベルスポンサーになった</li>
  <li>執筆活動全般</li>
  <li>Python EDでデータ分析試験を作ったときの裏話
    <ul>
      <li>合格できるレベルとはどんな人物像？</li>
      <li>主教材選びの難しさ</li>
      <li>マニアックになりすぎないような範囲にしたいと思った</li>
      <li>どう勉強したら良いか？</li>
      <li>オープンデータで具体的な分析をしてみると、勉強になるが、面白いデータを見るけるのは難しい</li>
    </ul>
  </li>
</ul>
